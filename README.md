# Alarm_Project

A simple Alarm project implemented by using while loop and if statements.

**I. Description**

This is simple alarm project to understand the working. Here, the program takes user input and displays the time. If the condition is satisfied
mp3 file is played, to notify the user that given input is matched with alarm time.


**II. Technology Stack**

Language: Python(3.6)

IDE used: Pycharm


**III. Workflow**

1. Firstly, program takes the user input, at what time he/she wants to set the alarm.
2. Hour, Minute and Am/PM should be choosen by the he/she to set the alarm.
3. A alarm time is displayed after the alarm is set.
4. If condition is applied to check for Am/Pm.
5. While loop is applied, if the given input is matched with alarm time mp3 file is played to report the user.
